# Real Estate Price Prediction

This project predicts real estate prices using advanced machine learning models like **XGBoost and LightGBM**. It includes:
- Data Preprocessing & Feature Engineering
- Exploratory Data Analysis (EDA)
- Feature Scaling & Transformation
- Model Training & Evaluation
- Feature Importance Analysis using SHAP & Permutation Importance

##  Files in this Repository:
- **`real_estate_price_prediction.py`** → The complete Python script for training and evaluating the model.
- **`DATA2.csv`** → The dataset used for model training.

##  How to Run the Code:
1. Download `real_estate_price_prediction.py` and `DATA2.csv`.
2. Run the Python script in **Jupyter Notebook, VS Code, or Google Colab**.
